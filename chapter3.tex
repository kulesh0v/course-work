\chapter{Результаты численных экспериментов}\label{chap3}

В этой главе применим схему MPC из главы (\ref{chap2}). Рассмотрим систему, которая принимает вид
\begin{displaymath}
	x_{k+1}=\begin{bmatrix}
		0.921 && 0 && 0.041 && 0 \\
		0 && 0.918 && 0 && 0.033 \\
		0 && 0 && 0.924 && 0 \\
		0 && 0 && 0 && 0.937
	\end{bmatrix}x_k+
	\begin{bmatrix}
		0.017 && 0.001 \\
		0.001 && 0.023 \\
		0 && 0.061 \\
		0.072 && 0
	\end{bmatrix}u_k
\end{displaymath}
\begin{displaymath}
	y_k=\begin{bmatrix}
		1 && 0 && 0 && 0\\
		0 && 1 && 0 && 0
	\end{bmatrix}x_k
\end{displaymath}
Я не буду рассматривать никаких ограничений на ввод или вывод. В эксперименте с разомкнутым контуром измеряется траектория ввода-вывода длиной N = 400, где вход выбирается случайным образом из единичного интервала, т.е. $u^d_k \in [-1,1]$. Также выберу $L=30$ для горизонта прогнозирования.
